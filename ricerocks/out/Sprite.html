Sprite

Each Sprite has these 15 common properties
Properties:
Name Type Description
Sprite.type String "spaceship" | "asteroid" | "missile" | "explosion"
Sprite.image HTMLImageElement initialised and loaded sprite sheet
Sprite.width pixels pixel width of a sprite tile, which is not necessarily image.width
Sprite.height pixels pixel height of a sprite tile, which is not necessarily image.height
Sprite.row Number offset sx is calculated from row * width, ie row = 0 for first tile
Sprite.column Number offset sy is calculated from column * height, ie column = 0 for first tile
Sprite.xCentre pixels from 0 in top left corner positive right, horizontal centre co-ordinate of sprite
Sprite.yCentre pixels from 0 in top left corner positive down, vertical centre co-ordinate of sprite
Sprite.xDelta pixels Sets horizonal speed of the sprite. xCentre is incremented by xDela each tick
Sprite.yDelta pixels Sets vertical speed of the sprite. yCentre is incremented by yDelta each tick
Sprite.angle radians radians to rotate image. -Math.PI/2 is 90 degrees
Sprite.angleDelta radians increment in radians to angle each tick, clockwise positive
Sprite.tick Number counter to select sprite tile and measure sprite age
Sprite.lifespan Number used to filter dead ephemereal sprites
